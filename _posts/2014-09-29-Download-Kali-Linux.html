Kali Linux adalah Linux berbasis distro Debian yang merupakan salah satu distro Linux yang sering digunakan untuk penetration testing. Dulunya distro ini bernama BackTrack Linux dan berbasis distro Ubuntu.

Kali Linux
Kali Linux biasanya sering dipakai oleh para hacker ataupun security specialist untuk menguji-coba keamanan sistem network atau server yang mereka miliki. Atau bahkan bisa digunakan untuk menyerang (hack) sistem network atau server milik orang lain.

Distro Penetration Testing Paling Canggih

Dari pencipta Kali Linux, beliau mengklaim bahwa Kali Linux merupakan distro Linux untuk penetration testing paling canggih yang pernah dibuat. Kali Linux memiliki seluruh rangkaian fitur luar biasa yang digunakan untuk membuat penetration testing menjadi lebih praktis.

Dukungan Hardware bagi ARM Fanatik

Bagi ARM Fanatik, terdapat Kali Linux (images) yang mendukung hardware Raspberry Pi, Odroid, Cubox, Beaglebones, Chromebooks. Pencipta Kali Linux terobsesi dengan mendapatkan Kali pada banyak mesin ARM menarik yang bisa mereka tangani, dan mereka punya cukup daftar hardware ARM yang impressive sejauh ini. Moto kami, ketika datang ke dunia ARM adalah "Kali all the things!".
