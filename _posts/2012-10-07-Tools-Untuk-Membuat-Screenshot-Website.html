--- layout: post title: Tools Untuk Membuat Screenshot Website date: 2012-10-07 03:15:10 UTC updated: 2012-10-07 03:15:10 UTC comments: false categories: internet --- Pada kesempatan kali ini saya ingin membahas tentang beberapa tools yang bisa digunakan untuk membuat screenshot sebuah halaman web, baik itu berupa website ataupun blog. Pada pembahasan kali ini saya menggunakan plugin dari browser Mozilla Firefox dan website pembuat screenshot.

Dengan menggunakan plugin Firefox.
  1. Screengrab!
    Cara penggunaannya mudah, klik kanan pada window/tab yang ingin diambil screenshot-nya, lalu pilih Screengrab! > Save > Complete Page/Frame, lalu pilih di direktori mana gambar akan disimpan.
  2. Abduction!
    Cara penggunaannya cukup mudah, klik kanan pada window/tab yang ingin diambil screenshot-nya setelah itu pilih Save Page as Image, kemudian klik 2 kali untuk menseleksi halaman secara penuh, lalu klik menu Save Selection saat menu pop-up muncul.
  3. Picnik
    Cara penggunaannya juga mudah, cukup klik kanan pada window/tab yang ingin diambil screenshot-nya, lalu pilih Send To Picnik, maka gambar akan diproses di server Picnik, tunggu beberapa menit, maka screenshot akan selesai dibuat.

Menggunakan bantuan website pembuat screenshot. 
  1. Thumbalizr
    Untuk menggunakannya cukup masukkan url halaman web yang akan di ambil screenshot-nya, lalu klik Thumb It. Anda pun bisa memilih besarnya resolusi gambar yang diinginkan.


  2. WebShotsPro
    Untuk menggunakannya cukup masukkan url halaman web yang akan di ambil screenshot-nya, lalu klik get screenshot of this website.
Sekian, semoga bermanfaat, dan silahkan kalau ada yang mau menambahkan :D.
