--- layout: post title: Menjalankan SQL Query di phpMyAdmin date: 2012-10-07 03:10:52 UTC updated: 2012-10-07 03:10:52 UTC comments: false categories: programming --- Salah satu database software yang umum digunakan untuk mengelola suatu web database adalah phpMyAdmin. Untuk membuat dan mengelola tabel di phpMyAdmin sebenarnya kita sudah diberi kemudahan dengan cara memilih dan mengklik menu-menu yang ada. Tapi bagi Anda yang ingin belajar mengenai SQL Query seperti DDL (Data Definition Language) dan DML (Data Manipulation Language) untuk mengelola tabel secara manual, Anda juga bisa menggunakan phpMyAdmin. Berikut caranya :


  1. Bagi Anda yang menggunakan XAMPP misalnya, buka phpMyAdmin di URL http://localhost/phpmyadmin/, atau jika Anda menggunakan cPanel buka phpMyAdmin di kolom Databases > phpMyAdmin.

  2. Buat database baru di kolom Create new database, misal dengan nama example, lalu klik Create.

  3. Kemudian pada sidebar klik icon Query window (icon kedua dari sebelah kiri), maka window baru akan muncul.

  4. Pada window baru tersebut ketikkan saja SQL Query yang diinginkan, sebagai contoh di sini saya membuat sebuah tabel dengan nama mahasiswa dengan 4 buah field, yakni npm, nama, kelas, dan ipk dengan field npm sebagai Primary Key-nya. Jika sudah, klik Go.

  5. Jika Query yang dimasukkan sudah benar maka akan muncul notifikasi Your SQL query has been executed successfully.
