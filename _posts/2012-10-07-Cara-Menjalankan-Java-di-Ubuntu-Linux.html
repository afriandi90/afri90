Pada tutorial kali ini saya ingin menjelaskan tentang cara menjalankan bahasa pemrograman Java di Ubuntu Linux. Saat ini Java merupakan bahasa pemrograman terpopuler di dunia. Mengapa bisa demikian? Karena bahasa pemrograman Java merupakan bahasa pemrograman yang bisa dijalankan di berbagai platform dan sistem operasi sehingga dikatakan sebagai bahasa pemrograman yang cross platform.

Nah, untuk dapat menjalankan aplikasi Java di suatu platform atau sistem operasi maka diperlukan JRE (Java Runtime Environment) yang digunakan untuk mengeksekusi program Java dan JDK (Java Development Kit) yang digunakan untuk mengkompile program Java.


Pada tutorial kali ini saya menggunakan JRE dan JDK dari Sun Microsystems. Untuk menginstall JRE dan JDK di Ubuntu ikuti langkah berikut :
  • Pastikan komputer Anda terhubung dengan internet, kemudian install JRE menggunakan perintah berikut :

    $ sudo apt-get install sun-java6-jre

    Ukuran file JRE dan JDK ini cukup besar, jadi Anda diharapkan mempunyai koneksi internet yang agak cepat.
  • Langkah kedua, install JDK menggunakan perintah berikut :

    $ sudo apt-get install sun-java6-jdk
Setelah kedua aplikasi ini terinstall, sebenarnya Anda sudah bisa menjalankan program Java menggunakan Terminal, dan untuk editornya Anda bisa menggunakan Gedit. Akan tetapi cara ini mungkin masih membingungkan bagi para pemula (ya termasuk saya ini :D).

Oleh karena itu dipelukan lagi sebuah aplikasi editor bernama Geany. Apa itu Geany? Bagi yang belum tahu, silahkan lihat di sini. Intinya Geany adalah aplikasi editor ringan yang dapat digunakan untuk menjalankan berbagai macam bahasa pemrograman.
  • Untuk menginstall Geany gunakan perintah berikut :
    $ sudo apt-get install geany
  • Setelah selesai, buka Geany di Menu > Programming >Geany.



  • Setelah Geany terbuka, klik menu File > New (with Template) > pilih Java source file. Di situ sudah disediakan template dengan syntax Java, Anda bisa menulis program Java di situ.



  • Setelah program selesai ditulis, Save dan Compile program.



  • Terakhir klik Run program.




Akhir kata, selamat belajar Java :D
