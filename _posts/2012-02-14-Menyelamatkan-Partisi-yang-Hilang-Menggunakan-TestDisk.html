--- layout: post title: Menyelamatkan Partisi yang Hilang Menggunakan TestDisk date: 2012-02-14 06:52:04 UTC updated: 2012-02-14 06:52:04 UTC comments: false categories: linux --- Pernahkah Anda mengalami kehilangan sebuah partisi Harddisk? Hilangnya partisi Harddisk ini biasanya terjadi karena Partition Table yang terhapus, akibatnya partisi Harddisk Anda menjadi RAW. Isi dari partisi Harddisk RAW ini sebenarnya masih ada, namun karena Partition Table nya hilang maka Harddisk tidak dapat dibuka.


Baru saja saya mencoba menginstall OpenSUSE di harddisk eksternal saya, setelah saya yakin konfigurasi partisi harddisk untuk menginstall OpenSUSE sudah benar, langsung saja saya memulai instalasi. Seharusnya semua berjalan lancar, karena sebelumnya saya mencoba install Ubuntu dengan konfigurasi partisi yang sama dan berhasil. Konfigurasinya kurang lebih seperti berikut :
/dev/sdb      extended   boot     250GB
/dev/sdb1       ntfs                        230GB
/dev/sdb2       ext4           /         20GB
Konfigurasi di atas hanya ilustrasi, harddisk saya partisi menjadi 2, pada partisi kedua (/dev/sdb2) saya letakkan / (root) dari Linux nya, dan pada MBR Harddisk (/dev/sdb) saya letakkan grub-loader nya. Sebelumnya konfigurasi seperti ini berhasil saat saya install Ubuntu, namun setelah saya install OpenSUSE partisi /dev/sdb1 berubah menjadi RAW, sehingga tidak dapat dibuka.


Tapi akhirnya partisi tersebut bisa saya selamatkan dan dapat dibuka kembali setelah saya mencoba menggunakan software yang bernama TestDisk. Software ini free dan berlisensi GPL v2+, enaknya lagi software ini tersedia untuk Linux, Windows, dan Mac OS. Untuk mendownload TestDisk silahkan merujuk ke halaman berikut :
Download TestDisk
Cara penggunaan TestDisk tidak terlalu sulit walaupun antar mukanya masih menggunakan Terminal/CMD jika di Windows. Tinggal ikuti step by step nya dari awal hingga akhir seperti yang terdapat di halaman berikut :
TestDisk Step by Step
Untuk lamanya proses recovery tergantung dari ukuran harddisk Anda, untuk merecovery 250GB software ini membutuhkan waktu sekitar 1 jam lebih.
