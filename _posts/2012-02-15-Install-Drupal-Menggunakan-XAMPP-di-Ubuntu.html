--- layout: post title: Install Drupal Menggunakan XAMPP di Ubuntu date: 2012-02-15 10:37:02 UTC updated: 2012-02-15 10:37:02 UTC comments: false categories: linux --- Pada tutorial kali ini saya ingin menjelaskan tentang cara install Drupal menggunakan XAMPP di Ubuntu. XAMPP merupakan sebuah program web server yang sudah sangat lengkap karena dalam sekali install kita bisa mendapatkan sebuah Apache Server, MySQL Database, dan tentu saja PHP. Dengan adanya XAMPP kita tidak perlu lagi repot-repot meng-install komponen tersebut satu-persatu. Karena menurut pengalaman teman saya jika ingin meng-install satu-persatu maka akan sangat menyulitkan saat mengkonfigurasi komponen-komponen tersebut. Baiklah, berikut ini akan saya jelaskan tutorialnya secara mendetail :

Langkah 1
Download XAMPP dan Drupal.
Download file yang diperlukan terlebih dahulu yakni XAMPP dan Drupal. Pastikan Anda mendownload versi stabil yang terbaru.


Langkah 2
Menginstal XAMPP.
Setelah selesai men-download file yang diperlukan, sekarang kita instal XAMPP terlebih dahulu. Pastikan file hasil download tadi satu folder dengan direktori kerja Anda.
  1. Misalkan Anda simpan file tadi di Desktop, maka jalankan perintah berikut di terminal. (login terlebih dahulu sebagai root) :
    # cd Desktop
  2. Ekstrak XAMPP ke direktori /opt :
    # tar xvfz xampp-linux-1.7.2.tar.gz -C /opt
  3. Kemudian jalankan XAMPP dengan perintah berikut :
    # /opt/lampp/lampp start
  4. Test apakah XAMPP sudah dapat bekerja.
    Buka browser Mozilla Firefox Anda, dan ketikkan :
    http://localhost
  5. Untuk menghentikan XAMPP gunakan perintah berikut :
    # /opt/lampp/lampp stop

Langkah 3
Mengekstrak Drupal.
  1. Ekstrak Drupal ke direktori opt/lampp/htdocs:
    # tar xvfz drupal-6.17.tar.gz -C /opt/lampp/htdocs
  2. Rename nama folder "drupal-6.17" menjadi "drupal":
    # mv /opt/lampp/htdocs/drupal-6.17 /opt/lampp/htdocs/drupal

Langkah 4
Menge-test Drupal
  1. Sekarang saatnya menge-test di komputer Anda. Buka browser Mozilla Firefox Anda, kemudian ketikkan :
    http://localhost/drupal/

Langkah 5
Membuat Database
  1. Buka tab baru di Mozilla Firefox Anda, kemudian ketikkan :
    http://localhost/phpmyadmin/
  2. Pada text box Create new database, ketikkan nama database yang diinginkan, misal "drupal".

Langkah 6
Meng-install Drupal
  1. Kembali ke tab sebelumnya di Mozilla Firefox Anda, kemudian klik link Install Drupal in English. Maka akan muncul halaman Requirements problem.
  2. Buat salinan dari file default.settings.php dan rename ke settings.php. Kembali ke Terminal dan ketikkan perintah berikut :
    # cp /opt/lampp/htdocs/drupal/sites/default/default.settings.php /opt/lampp/htdocs/drupal/sites/default/settings.php
  3. Berikutnya kita perlu memberikan permission ke file settings.php agar menjadi writable:
    # cd /opt/lampp/htdocs/drupal/sites
    # chmod a+w default
    # chmod a+w default/settings.php
  4. Terakhir kita perlu menambahkan PHP memory limit dari 8M ke 16M:
    # gedit /opt/lampp/etc/php.ini
    Tekan Ctrl + F, kemudian cari tulisan memory_limit dan ganti angkanya dari 8 ke 16.
  5. Kemudian restart XAMPP :
    # /opt/lampp/lampp stop
    # /opt/lampp/lampp start
  6. Kembali ke browser, kemudian klik link try again di tulisan paling bawah. Sekarang seharusnya Anda sudah bisa melihat halaman Database configuration di browser Anda.
    Masukkan nama database yang sebelumnya telah Anda buat, untuk database username isikan saja dengan "root", lalu klik Save and continue.
    Lalu Anda akan didirect ke halaman Configure site.
  7. Untuk tujuan keamanan, maka diberitahukan bahwa Anda harus menghapus permission writable di file settings.php:
    # chmod a-w /opt/lampp/htdocs/drupal/sites/default/settings.php
  8. Terakhir, masukkan informasi yang dibutuhkan, seperti Site information, dan Administration account. Jika sudah, klik Save and continue button.

Langkah 7
Menggunakan Drupal.
  1. Untuk mulai menggunakan Drupal, masuk ke halaman Admin :
    http://localhost/drupal/admin
  2. Kemudian masukkan username dan password Anda, lalu klik Log in.
