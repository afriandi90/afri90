--- layout: post title: Install OpenSSH Server di CentOS 6 date: 2012-10-26 02:58:51 UTC updated: 2012-10-26 02:58:51 UTC comments: false categories: linux networking --- OpenSSH (OpenBSD Secure Shell) adalah sebuah aplikasi yang digunakan untuk berkomunikasi secara secure di dalam jaringan komputer menggunakan protokol SSH. SSH sangat populer di pengguna Linux dan Unix varian lainnya. SSH adalah protokol utama yang digunakan untuk remote shell, sistem administrasi, pertukaran data, sinkronisasi dan secure tunnel.
SSH sebenarnya didesain untuk menggantikan Telnet (di port standar 23 TCP) yang dalam operasinya mengirimkan informasi dalam bentuk teks biasa tanpa enkripsi atau pengacakan. Sementara SSH dengan port standar 22 TCP, menyediakan enkripsi sebagai standar komunikasinya.
Berikut cara instalasi OpenSSH Server di CentOS 6 : Buka Terminal, masukkan perintah berikut :
$ sudo yum install openssh-server
Untuk menjalankan openssh-server masukkan perintah berikut :
$ sudo service sshd start
Kemudian konfigurasikan firewall di CentOS melalui menu System > Administration > Firewall untuk membuka port 22 (SSH) agar dapat diakses dari luar. Dari komputer lain kita dapat mengakses komputer remote yang telah diinstalasi OpenSSH Server ini melalui sintaks berikut di Terminal :
$ ssh username@host
Misalnya :
$ ssh afriandi@192.168.1.100
